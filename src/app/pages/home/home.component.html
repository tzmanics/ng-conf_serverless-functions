This project is the demo created for Frontend Master's Serverless Angular workshop. You can find the repo here on GitHub. If you want to see the project live, check it out at https://serverless-angular-workshop.netlify.app.

The following awesome tech was used to make this site:

  • Angular & Angular Universal for the frontend
  • Netlify platform & Netlify Functions
  • Sanity for the headless CMS
  • Stripe Checkout API for checkout and payment
  • Auth0 for login, auth & role checking

Phew, that's a lot of nice delegation and decoupling ;}
